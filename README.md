## how to launch

1. clone repo

2. `npm install`

3. create a .env file based on the example below

4. `node main.js`

## example .env

```
TOKEN="token here"
PREFIX="m="
OWNER="209765088196821012"
DEBUG=false
```

(if debug is true, debug logs will appear. enable this if you want to submit a bug with logs)
